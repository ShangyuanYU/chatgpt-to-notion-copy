# Privacy Policy for ChatGPT to Notion Copy

**ChatGPT to Notion Copy** is a Chrome extension designed to convert content copied from ChatGPT into a Notion-friendly Markdown format while preserving code blocks and mathematical formulas.

## Data Collection

This extension does **not collect, store, or transmit** any personal or usage data. All processing happens locally in your browser.

## Clipboard Access

The extension only accesses your clipboard content in response to explicit user actions, such as clicking the copy button. Clipboard content is **not stored or sent** anywhere.

## Host Access

The extension only runs on the ChatGPT website to process content that you copy. It does not access or interact with any other websites, and no browsing activity is monitored.

## Third-Party Services

No analytics, tracking, or third-party services are used in this extension.

## Local Processing

All functionality, including formatting preservation for code blocks and formulas, occurs entirely within your browser. Nothing is transmitted to external servers.
